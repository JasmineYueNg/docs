## How to Get Your Figma Access Token

<Warning>
  You need a Personal Access Token (PAT) to connect Figma. This token allows YouWare's MCP to securely access your design files via the Figma API.
</Warning>

### Step-by-Step Setup

<Steps>
  <Step title="Access Profile Settings">
    Click on your avatar in the top-left corner and select **Settings** from the dropdown menu.
  </Step>
  
  <Step title="Generate New Token">
    In the Settings page, select the **Security** tab and scroll down to the **Personal access tokens** section. Click the **Generate new token** button.
  </Step>
  
  <Step title="Configure Token Settings">
    <div className="space-y-3">
      <div>
        <strong>Token Name:</strong> Give your token a recognizable name, like <code>YouWare MCP</code>
      </div>
      <div>
        <strong>Expiration:</strong> We recommend setting the expiration to a reasonable period, such as 90 days
      </div>
      <div>
        <strong>Scopes:</strong> Set the following permissions:
        <ul className="list-disc list-inside mt-2 space-y-1">
          <li><code>file_content:read</code></li>
          <li><code>file_dev_resources:read</code></li>
          <li><code>projects:read</code></li>
        </ul>
      </div>
    </div>
  </Step>
  
  <Step title="Copy Token Immediately">
    <div className="p-4 bg-amber-50 border border-amber-200 rounded-lg">
      <div className="flex items-center gap-2 mb-2">
        <Clock className="h-4 w-4 text-amber-600" />
        <strong className="text-amber-800">Important:</strong>
      </div>
      <p className="text-amber-700">
        Once the token is generated, copy it right away — you won't be able to see it again. Make sure to copy your personal access token now as you will only be able to see this once.
      </p>
    </div>
  </Step>
</Steps>

### How to Do It

<Steps>
  <Step title="Select Section">
    In Figma, select a specific section (header, hero, features, footer, etc.)
  </Step>
  
  <Step title="Copy Link">
    Right-click and choose "Copy link to selection"
  </Step>
  
  <Step title="Generate Code">
    Paste the link into YouWare and let the AI generate that section
  </Step>
  
  <Step title="Repeat Process">
    Repeat for each major section of your design
  </Step>
  
  <Step title="Assemble">
    The AI will assemble all sections into a cohesive, fully functional page
  </Step>
</Steps>

  <AccordionItem value="access-token">
    <AccordionTrigger>How do I get a Figma Personal Access Token?</AccordionTrigger>
    <AccordionContent>
      <ol className="list-decimal list-inside space-y-2">
        <li>Click your avatar in Figma and select <strong>Settings</strong></li>
        <li>Go to the <strong>Security</strong> tab</li>
        <li>Scroll to <strong>Personal access tokens</strong> and click <strong>Generate new token</strong></li>
        <li>Name it (e.g., "YouWare MCP"), set expiration (90 days recommended)</li>
        <li>Enable these scopes: <code>file_content:read</code>, <code>file_dev_resources:read</code>, <code>projects:read</code></li>
        <li>Copy the token immediately (you won't see it again!)</li>
      </ol>
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="permissions">
    <AccordionTrigger>What Figma permissions (scopes) do I need to enable?</AccordionTrigger>
    <AccordionContent>

        <li><code>file_content:read</code> - Allows reading design content</li>
        <li><code>file_dev_resources:read</code> - Allows reading development resources</li>
        <li><code>projects:read</code> - Allows reading project structure</li>
      </ul>
    </AccordionContent>
  </AccordionItem>
