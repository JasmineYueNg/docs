## Getting Started

<Note>
The MCP Tool entry point is located at the bottom left of the input box on the page creation screen.
</Note>

### Accessing Tools

1. **Current Tools**: Click the MCP Tool button to expand the list of available tools for the current project
2. **Browse Market**: Click the "+" button in the upper right corner to enter MCP Market and discover new tools

### Tool Authorization

Some tools (such as Figma and database-related tools) can be authorized and linked through simple guidance. These tools come with:

- **Configuration Instructions**: Step-by-step setup guides
- **Usage Examples**: Ready-to-use templates and workflows
- **Easy Integration**: Simple authorization process


## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How do I install new MCP tools?">
    Navigate to the MCP Market by clicking the "+" button in the tools panel. Browse available tools and follow the installation instructions for each tool.
  </Accordion>
  
  <Accordion title="Can I use multiple tools simultaneously?">
    Yes, MCP tools are designed to work together. You can chain multiple tools in a single workflow to create complex, integrated solutions.
  </Accordion>
  
  <Accordion title="Do I need to configure each tool separately?">
    Some tools require initial authorization (like Figma or database tools), but most come with simple setup guides and examples to get you started quickly.
  </Accordion>
  
  <Accordion title="Are there limitations on tool usage?">
    Tool limitations depend on the specific service and your subscription plan. Check individual tool documentation for usage limits and pricing information.
  </Accordion>
</AccordionGroup>
